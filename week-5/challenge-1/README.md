# Challenge 1 - 글 내용에 추신 달기

## 문제

### 글을 쓰면 자동으로 `- 추신: 감사합니다.` 가 나오게 만들어봅시다.

어떤 내용으로 글을 작성하더라도, 글의 내용 마지막에 `- 추신: 감사합니다.`가 따라나오게 만들어 봅시다. 즉 글 내용에 추신을 첨가하여 저장시킵니다.
