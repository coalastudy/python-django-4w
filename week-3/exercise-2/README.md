# 자율과제 2 - 배수판단 파이썬 프로그램

## 문제

### 숫자 21과 22가 7의 배수인지 판단하는 Python 프로그램을 작성하세요.

{% hint style="info" %}
### 배수란?

배수란 어떤 수에 정수를 곱한 수를 의미합니다. \(즉 그 수에 의해 나누어 떨어짐\) e.g 10은 5의 배수
{% endhint %}

{% hint style="info" %}
### 파이썬에서 배수를 체크하는 방법

`X % N`이 0이면?  
나머지가 0이므로, X는 N의 배수입니다.
{% endhint %}
